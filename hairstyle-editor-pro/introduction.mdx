## 计费说明

### 计费单位

- **API 调用次数**：每次成功的 API 请求计为一次调用
- **失败请求**：失败的 API 请求不计费
- **异步任务**：异步任务按提交次数计费，不按查询次数计费

### 计费周期

- **实时计费**：API 调用后立即计费
- **余额不足**：账户余额不足时 API 调用将被拒绝

## 成本优化建议

### 1. 合理使用异步处理

- 对于不紧急的任务，使用异步处理可以降低成本
- 异步处理通常比同步处理更便宜

### 2. 批量处理

- 将多个请求合并为批量请求
- 批量处理享受折扣优惠

### 3. 图片预处理

- 在上传前对图片进行压缩和优化
- 减少文件大小可以降低传输成本

### 4. 缓存策略

- 对相同参数的请求结果进行缓存
- 避免重复调用相同的 API
