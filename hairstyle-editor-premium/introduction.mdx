# 发型设计专业版

发型设计专业版用于生成更精细、更可控的发型效果。您可以直接选择系统预设发型，也可以上传参考图，让模型根据参考图迁移发型；发色既可以使用预设颜色，也可以跟随参考图。

<Tip>
如果只需要基础的预设发型和预设发色，可以继续使用[发型编辑 API](/api-reference/hairstyle-editor-pro)。如果需要参考图发型迁移或参考图发色，请使用专业版。
</Tip>

## 核心能力

| 能力 | 说明 |
|------|------|
| 预设发型 | 通过 `hair_style` 选择内置发型模板。 |
| 参考图发型迁移 | 通过 `image_template` 上传参考图，将参考图中的发型迁移到主图人物。 |
| 参考图发色 | `color` 使用 `reference` 时，发色跟随参考图。 |
| 预设发色 | 支持金发、棕发、蓝发、粉发、黑发、银发、渐变发等多种发色。 |
| 异步处理 | 提交任务后返回 `task_id`，再通过异步任务查询接口获取结果。 |

## 专业版与基础版对比

| 项目 | 发型编辑 API | 发型设计专业版 API |
|------|------|------|
| 预设发型 | 支持 | 支持，发型选项更多 |
| 参考图发型 | 不支持 | 支持 |
| 参考图发色 | 不支持 | 支持 |
| 图片大小 | 不超过 3MB | 不超过 5MB |
| 图片分辨率 | 大于 200x200px，小于 1999x1999px | 大于 200x200px，小于 4090x4090px |
| 最小面部比例 | 20% | 10% |
| 单次费用 | 按基础版计费 | ￥0.15/次 |

## 计费说明

- **单次费用**：每次成功提交专业版任务计费 `￥0.15`。
- **计费时机**：异步任务按提交次数计费，不按查询次数计费。
- **失败请求**：请求失败不计费。
- **余额不足**：账户余额不足时 API 调用会被拒绝。

## 使用流程

1. 准备主图，确保图片中包含清晰完整的人脸。
2. 选择一种发型来源：
   - 使用 `hair_style` 选择内置发型；
   - 或上传 `image_template` 作为参考图。
3. 选择发色：
   - 使用 `reference` 跟随参考图发色；
   - 或传入预设发色值。
4. 调用[发型设计专业版 API](/api-reference/hairstyle-editor-premium) 提交任务。
5. 保存返回的 `task_id`，调用[异步任务查询 API](/api-reference/async-task-results) 获取结果。

## 图片要求

- **图片格式**：`PNG`、`JPG`、`JPEG`
- **图片大小**：不超过 5MB
- **图片分辨率**：大于 200x200px，小于 4090x4090px
- **最小面部比例**：图像中面部比例不能小于 10%
- **面部完整性**：建议使用面部无遮挡、五官清晰的照片
- **面部角度**：建议使用正面或接近正面的照片

<Warning>
上传参考图时，请选择发型轮廓清晰、遮挡较少的人像照片。参考图质量越高，发型迁移结果越稳定。
</Warning>

## 文件存储策略

<Tip>

- **上传文件**：不存储。
- **响应文件**：保留 24 小时，超时后自动删除。

</Tip>

## 应用场景

- **美发试发型**：用户上传自拍并参考明星、模特或样片发型生成试发效果。
- **虚拟形象设计**：为头像、写真、虚拟人物生成统一风格的发型。
- **电商和美业工具**：在预约、导购、试妆流程中提供更高可控性的发型预览。
- **内容创作**：批量生成不同发型、发色方案，用于社交媒体和营销素材。
