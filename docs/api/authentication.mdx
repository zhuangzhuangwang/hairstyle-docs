# 认证方式

AILabTools API使用API密钥进行身份验证和授权。

## 获取API密钥

### 1. 注册账户

访问 [AILabTools官网](https://www.ailabtools.com/account/signup) 注册您的账户。

### 2. 登录控制台

登录后进入 [控制台](https://console.ailabtools.com) 获取您的API密钥。

### 3. 生成API密钥

在控制台的"API密钥"页面，您可以：
- 查看现有的API密钥
- 生成新的API密钥
- 删除不再使用的API密钥

<Warning>
API密钥是访问所有API的凭证，请妥善保管，不要泄露给他人。如果密钥泄露，请立即在控制台中重新生成。
</Warning>

## 使用API密钥

### 请求头认证

在所有API请求的请求头中包含您的API密钥：

```bash
Authorization: Bearer YOUR_API_KEY
```

### 示例

<Tabs>
<Tab title="cURL">
```bash
curl -X POST https://api.ailabtools.com/v1/hairstyle-editor-pro \
  -H "Authorization: Bearer sk-1234567890abcdef" \
  -H "Content-Type: application/json" \
  -d '{
    "image": "data:image/jpeg;base64,/9j/4AAQSkZJRgABAQAAAQ..."
  }'
```
</Tab>

<Tab title="JavaScript">
```javascript
const response = await fetch('https://api.ailabtools.com/v1/hairstyle-editor-pro', {
  method: 'POST',
  headers: {
    'Authorization': 'Bearer sk-1234567890abcdef',
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    image: 'data:image/jpeg;base64,/9j/4AAQSkZJRgABAQAAAQ...'
  })
});
```
</Tab>

<Tab title="Python">
```python
import requests

response = requests.post(

    headers={
        'Authorization': 'Bearer sk-1234567890abcdef',
        'Content-Type': 'application/json'
    },
    json={
        'image': 'data:image/jpeg;base64,/9j/4AAQSkZJRgABAQAAAQ...'

```
</Tab>

## 安全最佳实践

### 1. 环境变量

不要在代码中硬编码API密钥，使用环境变量：

```bash
# .env 文件
AILABTOOLS_API_KEY=sk-1234567890abcdef
```

```javascript
// JavaScript
const apiKey = process.env.AILABTOOLS_API_KEY;
```

```python
# Python
import os
api_key = os.getenv('AILABTOOLS_API_KEY')
```

### 2. 密钥轮换

定期轮换您的API密钥：
1. 在控制台生成新的API密钥
2. 更新您的应用程序配置
3. 删除旧的API密钥

### 3. 权限管理

- 为不同的应用程序使用不同的API密钥
- 定期审查API密钥的使用情况
- 及时删除不再使用的API密钥

## 错误处理

### 认证失败

当API密钥无效或缺失时，API会返回401状态码：

```json
{
  "request_id": "req_123456789",
  "log_id": "log_987654321",
  "error_code": 2001,
  "error_detail": {
    "status_code": 401,
    "code": "AUTHENTICATION_FAILED",
    "code_message": "认证失败",
    "message": "API密钥无效或已过期"
  }
}
```

### 权限不足

当账户没有访问特定API的权限时，会返回403状态码：

```json
{
  "request_id": "req_123456789",
  "log_id": "log_987654321",
  "error_code": 2002,
  "error_detail": {
    "status_code": 403,
    "code": "INSUFFICIENT_PERMISSIONS",
    "code_message": "权限不足",
    "message": "当前账户没有访问该API的权限"
  }
}
```
