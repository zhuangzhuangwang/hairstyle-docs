# 获取API密钥

AILabTools是一个先进的工具，提供大量简单灵活的API端点来满足您的特定需求。只需一个API密钥，您就可以轻松调用任何端点，并快速集成到您的应用程序或工作流程中。

## 获取步骤

### 第一步：注册账户
<a href="https://www.ailabtools.com/account/signup" target="_blank">立即注册</a>

### 第二步：登录开发者平台
<a href="https://www.ailabtools.com/developer-platform" target="_blank">登录开发者平台</a>

### 第三步：完成邮箱认证
- 这里的认证邮箱与登录邮箱无关
- 如果之前认证过邮箱，可以直接进入下一步

### 第四步：创建应用
在开发者平台创建新的应用

### 第五步：获取API密钥
创建应用后即可获取API密钥

<Warning>
API密钥是访问所有API的凭证，请妥善保管，不要泄露给他人。
</Warning>
