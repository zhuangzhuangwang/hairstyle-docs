# 快速开始

欢迎使用 AI发型设计 API！本指南将帮助您快速上手并开始使用我们的 AI 服务。

## 准备工作

### 1. 获取 API Key

首先，您需要获取一个 API Key：

1. 访问 [AI发型设计 开发者中心](https://www.aihairstyle.cn/developer-platform) 注册账号
2. 登录开发者平台
3. 创建应用并获取 API Key

<Warning>
  API密钥是访问所有API的凭证，请妥善保管，不要泄露给他人。
</Warning>

- **步骤1**: <a href="https://www.aihairstyle.cn?source=dev" target="_blank">注册账户</a>

  如果您已经有AI发型设计账户，可以直接进行下一步。

- **步骤2**: <a href="https://www.haristyle.cn/developer-platform" target="_blank">前往开发者中心</a>

  成功登录后您将获得指定数量的积分（仅支持用于API调用）。

  ![注册即送](/images/dev/1.webp)


- **步骤3**: 创建应用

  ![创建应用](/images/dev/2.webp)

- **步骤4**: 获取API密钥

  ![获取API密钥](/images/dev/3.webp)
